Resume-Demo
-----------
### Synopsis
Resumes a Demo

---
### Description

Resumes a Demo that was paused or debugged with `!`.

---
### Related Links
* [Show-Demo](Show-Demo.md)



* [Get-Demo](Get-Demo.md)



---
### Examples
#### EXAMPLE 1
```PowerShell
Resume-Demo
```

---
### Parameters
#### **DemoToResume**

The demo that will be resumed.



> **Type**: ```[Object]```

> **Required**: false

> **Position**: 1

> **PipelineInput**:true (ByValue)



---
### Syntax
```PowerShell
Resume-Demo [[-DemoToResume] <Object>] [<CommonParameters>]
```
